How to install the JobSeeker web service :

1) install nodejs version 7 minimum and also npm.
2) install and use PostgreSQL to create a database called "service"
3) install the web service.
4) install the project's dependancies running "npm install" in the software folder.
5) change in settings.json the username and password for the database;
6) run a first time the app with "node .", the database should be initialized.
